2.3.7 HTML的图片元素

在HTML中，图像是由<img>元素定义的，<img>元素是空元素标签， 也就是说， 了更加严谨和可靠，在实际开发中，最好写成<img/>。

有很多情况，一张图片可能胜过千言万语，但是图片过多或者过大， 也可能造成用 的等待，其至造成用户不知所云。所以，在编写HTML文档时，图文并茂一定要合理。

<img>元素的基本结构如下：

<img src=图片的 urlalt=图像的替代文本＂/>

图片的URL指存储图像的位置，可以是相对路径，也可以是绝对路径。

图像的替代文本主要针对的是无图浏览器，或者纯文字型浏览器， 图片会显示成图 的替代文本。

                
                
示例代码：
<!DOCTYPE HTML PUBLIC "-//m3c//DD BTML 4.01//EN" "http://W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
<html>
    <head>
        <meta http-eguiv="content-type" content="text/html" charset="utf-8">
        <title>图片</title>
    </head>
    <body>
        <img src="https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png" alt="Baidu Logo">
    </body>
</html>

<img>元素还有以下一些属性可以使用。

·height:设置图片的高度。

·width:设置图片的宽度。

            
            
示例代码：
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/htm14/strict.dtd">
<html>
    <head>
        <meta http-equiv="content-type" content="text/html"; charset="utf-8">
        <title>图片</title>
    </head>
    <body>
        <img src="https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png" alt="Baidu Logo" width="117" height="38"><br />
        <img src="https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png" alt="Baidu Logo" width="234" height="76"><br />
        <img src="https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png" alt="Baidu Logo" width="351" height="114">
    </body>
</html>
